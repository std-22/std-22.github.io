Company strategy

Данный сайт является визитной карточкой компании Studio 22.

На нём размещена информация о креативном пространстве, нашей команде и реализуемых проектах.

Основной задачей сайта является привлечение новой аудитории.

Сайт написан при помощи языка разметки гиертекста HTML, формального языка описания внешенего вида документа CSS и языка программирования JavaScript.

В работе был задействован фреймворк CSS Grid, благодаря которому сайт стал адаптивен на мобильных устройствах.

Наша команда благодарит Github за предоставленную возможность бесплатного размещения сайта в сети интернет при помощи Github Pages.

В разработке сайта принимали участие: Меликбекян Ашот, Лёвин Александр и Толоконов Илья.

Предстоящие задачи

  • По мере развития компании будут открываться новые возможности, которыми мы будем делиться здесь, создавая новые странницы, галереи и многое другое.

  • Мы ждём новые лица и реализовываем новые проекты, которые впоследствии также окажутся на этом сайте.

  • Яндекс-метрика позволяет анализировать информацию о наших пользователях, а значит там можно найти подсказки о том, как улучшить нашу работу.

  • Сайт - очень удобный способ представления практически любой информации. Можно попробовадь сделать свой интернет-магазин или мини-игру, так зачем медлить?
